发表于 2024-11-03

What is 迭代器：

迭代器（iterator）是一种用于遍历集合（如数组、向量、链表等）的抽象机制。

Iterator Trait 定义：

Iterator trait 是 Rust 标准库中定义的一个重要 trait，它有两个主要部分：

  1. **关联类型 type Item**：表示迭代器产生的项的类型。
  2. **方法 fn next(&mut self) -> Option<Self::Item>**：返回集合中的下一个元素。如果集合中没有更多元素，则返回 None。
